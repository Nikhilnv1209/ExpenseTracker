# Expense Tracker

A modern Android expense tracking app built with Jetpack Compose and Material Design 3.

## Features

- **Dashboard** — Greeting header, total balance card, income/expense stats
- **Calendar View** — Monthly grid with daily expense bar indicators
- **Line Chart** — Weekly spending chart with smooth cubic bezier curves
- **Cal/Chart Toggle** — Switch between calendar and chart views
- **Dark Theme** — Violet accent with dark gray background, full edge-to-edge

## Tech Stack

- **Language:** Kotlin 2.0.21
- **UI:** Jetpack Compose + Material 3
- **Build:** Gradle 9.2.1 + Android Gradle Plugin 9.0.1
- **Architecture:** Single-activity, Compose-first
